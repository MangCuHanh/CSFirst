# Vòng lặp WHILE

Cách viết
```python
while [điều kiện đúng]:
    print()
```

vidu
```python
so_lan_lap = 6
while so_lan_lap > 0:
    so_lan_lap -= 1
    print(so_lan_lap)

end = 0
while not at_goal():
    jump()

while at_goal != True:
    jump()

while 5 > 3:
    print(5 > 3)    
```
